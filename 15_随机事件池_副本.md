## 触发机制

### 基本规则

每次进入副本后，队伍穿行于 2~5 层地牢。**每一层进入时**进行事件判定：

| 参数 | 说明 |
|------|------|
| **战斗/事件比** | 非BOSS层：**45% 战斗 / 55% 事件**。BOSS层：100% BOSS战，不触发通用事件 |
| **事件类别分布** | 宝箱 20% \| 陷阱 15% \| 岔路 15% \| 奇遇 10% \| 休息点 10% \| 精英怪 15% \| 空层 15%（合计100%） |
| **域事件覆盖** | 在五域副本中，抽取通用事件后有 **30%** 概率被替换为该域的专属事件（同类别替换，如陷阱→域陷阱） |
| **首层保护** | 副本第1层必定为**战斗**（不会触发事件），以建立区域氛围和检验队伍强度 |
| **岔路残留** | 岔路事件的选择会影响**下一层**的事件池权重（见岔路类各条目详述），效果仅持续1层 |
| **重复抑制** | 同一事件在同一次副本内不会出现第二次；精英怪事件同副本最多触发1次 |
| **难度系数** | 困难难度：精英怪属性×1.2，陷阱伤害×1.3。噩梦难度：精英怪属性×1.5，陷阱伤害×1.6，所有负面选项额外附加1层随机 DOT |

### 与副本结构的整合

```
进入副本 → 第1层强制战斗（暖场）
       ↓
第2层 → roll: 战斗 or 事件 → 若事件，roll类别+具体事件
       ↓
第3~4层 → 同上（若副本仅2层则跳过）
       ↓
末层（BOSS层） → BOSS战，无通用事件
       ↓
副本结束 → 清除副本内状态（专属资源清零、DOT清除）
```

### 副本内状态说明

- **丹药/道具**：副本中可使用已携带的丹药和道具，消耗后从背包扣除
- **专属资源**（剑气/道印/药力/正气/血煞/蛊毒层/妖力/兽威）：在连续战斗中**跨战斗保留**，副本结束后清零
- **DOT/控制**：战斗结束后清除（瘴毒、冰冻、缠绕等不跨战斗持续）
- **站位**：战斗外可自由调整站位；某些事件可能临时改变站位，进入下一场战斗时生效
- **HP/道心**：战斗结束后HP不自动回复（除非触发休息点事件或使用丹药）；道心在副本内按正常规则波动

### 宝箱类（替代原"20%宝箱"）

#### 事件：灵光宝箱
**描述**：甬道尽头，一口青铜箱半埋在碎石中，缝隙间渗出的灵光将石壁映得幽幽发蓝。箱盖上刻着已经模糊不清的封印符——不知是镇压箱中之物，还是保护它免于外人之手。

**选项**：
1. 开箱取宝 — **结果**：获得灵石+封存物。80%概率获得灵石×60~150 + 随机白色/绿色装备×1；15%概率触发残存封印，全队受到ATK×30%的灵力冲击伤害；5%概率箱中空无一物，只有一张泛黄字条：「上一个开箱的，已经死了。」
2. 先试探再决定 — **结果**：消耗1回合检测。箱子若为陷阱（若有），概率降低至5%。但检测期间的灵力波动吸引了游魂×2，需立即进入战斗（若本层本应是事件则改为战斗，战斗后仍可获得宝箱内容物）。
3. 不碰，绕行 — **结果**：安全通过，无事发生。但队伍中若有幽冥血宗或万毒门成员，其道心-3（「你在怕什么？」）。

#### 事件：上锁秘箱
**描述**：一具枯骨斜靠在墙边，怀中紧抱一口精铁小箱。枯骨的手指仍扣在箱锁上，指节已碎。锁孔里塞着半截断掉的钥匙——前人打不开，至死不肯放手。

**选项**：
1. 暴力破拆 — **结果**：全队进行一次判定（取队伍最高ATK vs 难度150）。成功→箱中藏有灵石×120 + 随机蓝色品质材料×1（如银矿石、天蚕丝等）。失败→触发机关毒针，前排2人各受ATK×50%的毒属性伤害+瘴毒1层（持续至下场战斗结束）。
2. 让巧手之人开锁 — **结果**：需队伍中有SPD≥15的角色。成功开锁→获得灵石×100 + 随机紫色品质材料×1（如金精石、万年冰髓等）。开锁者道心+5（「技术在身，饿不死」）。
3. 连箱子带枯骨一起埋了 — **结果**：浩然书院或悬壶医谷成员道心+8。枯骨怀中滑落一枚旧玉佩（饰品·木制护符），以及一封未寄出的信。信中提到某处「地下第三间石室藏着续骨丹×2」——若本次副本为西域，下一层必定额外出现一个宝箱事件。

#### 事件：双生宝箱
**描述**：石室中央并排摆着两口一模一样的箱子——同样的雕纹、同样的铜绿、同样的尺寸。左侧箱盖上用血写着「开」，右侧写着「不开」。字迹新鲜，还在往下淌。

**选项**：
1. 开左侧（「开」） — **结果**：箱中冒出腐臭黑烟，全队获得瘴毒1层（持续至下场战斗结束）。但箱底压着灵石×200 + 解毒丹×2。写字的血还在——那人在用毒烟逼退后来者，好独占箱中物，但他死在了另一条路上。
2. 开右侧（「不开」） — **结果**：箱中安静地躺着一件蓝色品质装备 + 灵石×80。没有机关，没有毒烟。写字的人果然在骗你。但当你拿起装备时，发现箱底刻着一行小字：「骗你的人是我。但开右边的人，也未必能活着走出这个副本。」——下一层精英怪出现概率+20%。
3. 两口都开 — **结果**：先触发左侧毒烟（全队瘴毒2层），再获得右侧装备。额外在两口箱子夹缝中发现前人的遗物——随机获得丹药×2（回血丹/续骨丹/狂暴丹中随机）。贪婪从来都有代价，但也从来都有回报。

### 陷阱类（替代原"15%陷阱"）

#### 事件：毒瘴喷口
**描述**：地道两侧的石壁上密布着拳头大的孔洞，洞口边缘泛着荧光绿的结晶体。脚下一滑——你踩碎了一枚早已腐朽的虫卵，清脆的碎裂声在甬道中回荡。下一秒，所有孔洞同时嘶嘶作响。

**选项**：
1. 全速冲刺 — **结果**：全体进行一次SPD判定（取队伍平均SPD vs 难度120）。成功→穿过毒瘴区，仅后排3人受到瘴毒1层。失败→全体瘴毒2层（持续至下场战斗结束），且后排5号位额外中毒（「跑在最后的那个，永远最惨」）。
2. 用灵力封堵喷口 — **结果**：需队伍中有太乙道宗成员（消耗道印×1）或悬壶医谷成员（消耗药力×1）。成功封堵→安全通过，且获得瘴气结晶×3（封堵过程中凝结的结晶）。若无相应门派成员，则改为消耗回血丹×2敷住喷口→效果相同但消耗丹药。
3. 退回去找另一条路 — **结果**：绕路而行。本层不会遭遇战斗，但下一层必定触发一场战斗（怪物因你绕路而有了集结时间）。绕路途中发现前人遗落的灵草×2。

#### 事件：塌陷地板
**描述**：前方的石板路看上去平淡无奇，但队伍中最轻的那人踏上去时——石板微微下陷了一寸。低头看去，石板缝隙间隐约可见下方是空的，深处传来若有若无的流水声。这下面是地下暗河，还是更糟的东西？

**选项**：
1. 依次轻步通过 — **结果**：取队伍最低体重角色先行试探。进行一次SPD判定（最低SPD vs 难度100）。成功→全队安全通过。失败→体重最大的前排角色踩塌石板，坠入下层——前排1号位受到ATK×60%的坠落伤害+断骨（SPD-5，持续至副本结束）。但坠落者在下层发现了金沙×5 + 古币×3（西域副本中改为获得古国法器残片×1）。
2. 铺板架桥 — **结果**：消耗时间搭建临时桥面。无风险通过，但耗时使下一层战斗时敌方先手+1回合（突袭效果）。架桥过程中，有成员发现石壁上嵌着一枚旧铜铃——获得铜铃铛×1。
3. 干脆跳下去探底 — **结果**：全队跃入下层。全体受到坠落伤害（ATK×30%）。下层是一条被遗忘的排水暗道，没有怪物但也没有出口——只能沿暗河前进，直接跳过本层，进入下一层。暗河尽头漂着一具穿紫绶仙衣的骸骨→获得紫绶仙衣×1（但需丢弃一件当前防具才能穿上，因为「尸水泡过的衣服，得先用灵火炼化」——若队伍无火属性技能，则获得材料天蚕丝×2代替）。

#### 事件：机关箭阵
**描述**：走廊两侧的墙壁上整齐排列着密密麻麻的箭孔，地板上有十几具骷髅——每具都面向出口方向，后背插满箭矢。他们不是冲不过去，是冲到一半时发现出口也是假的。

**选项**：
1. 举盾推进 — **结果**：前排1号位承受全部箭矢伤害（ATK×100% + 破甲状态：DEF-20%，持续至下场战斗结束）。后排安全通过。需队伍中有前排角色（1或2号位有活人），若前排无人则改为全体各承受ATK×60%伤害。
2. 观察箭孔规律 — **结果**：进行一次SPD判定（取队伍最高SPD vs 难度130）。成功→找到射击间隙，全队无伤通过，且发现箭矢由某种精铁铸造→获得铁矿石×5。失败→触发双重机关——除箭矢外，地板翻转，全队落入陷阱坑，受到ATK×40%伤害+瘴毒1层。
3. 用尸体挡箭 — **结果**：消耗1具尸体（若队中无人阵亡则不可选）。将尸体推入走廊触发全部机关，待箭矢耗尽后安全通过。幽冥血宗成员道心+3（「死人就该这样用」），但浩然书院或悬壶医谷成员道心-5（「连死者都不放过，还算是人吗」）。通过后在箭孔后方的机关室中发现灵石×100 + 随机白色武器×1。

#### 事件：幻阵迷途
**描述**：走着走着，前方的路分成了三条——但一眨眼又变回了一条。再眨眼，是五条。墙壁在呼吸，石砖的纹理像血管一样蠕动。你回头看去，来路已经消失，取而代之的是一面刻满扭曲人脸的墙。其中一张脸，长得像你自己。

**选项**：
1. 集中心神破解幻阵 — **结果**：全体进行一次道心判定（取队伍平均道心 vs 难度100）。成功→幻阵破除，发现不过是三块破旧的迷魂幡插在角落。获得迷魂幡碎片→可出售灵石×50。太乙道宗成员额外获得道印×2（「区区幻术，不过尔尔」）。失败→全体陷入混乱1回合（下场战斗开始时生效），且道心-5。
2. 闭上眼睛走直线 — **结果**：无视幻象，摸墙前行。无伤害通过，但全队在下场战斗中命中率-15%（幻阵余波影响）。前行中摸到墙上一处暗格→获得清心丹×1。
3. 攻击墙壁上的人脸 — **结果**：前排角色对墙攻击。幻阵受到刺激后剧烈反噬——全队道心-10，且随机1名成员获得「恐惧」状态（下场战斗中每回合有15%概率跳过行动）。但暴力破坏了幻阵核心——获得灵石×150 + 镇魂丹×1。幻阵破碎时，那面人脸上的你，对你笑了一下。

### 岔路类（替代原"15%岔路"）

> **核心设计**：以下岔路选择不仅是文字叙事，会对后续层产生实际的**概率偏移**和**内容变化**。效果标注于每个选项末尾。

#### 事件：双通道抉择
**描述**：面前分出两条截然不同的通道。左路狭窄潮湿，岩壁上爬满散发荧光的苔藓，滴水声在深处回响——这条路显然年久失修。右路宽阔干燥，地面有整齐的铺砖，两侧墙壁上甚至还有熄灭的灯台——像是被维护过的通道。但问题恰恰在此：在这个地方，有什么东西在维护一条通道？

**选项**：
1. 走左路（窄湿旧路） — **结果**：陷阱概率高，但怪物少。**下一层事件池偏移**：陷阱类权重×2，战斗触发概率-15%。狭窄处发现前人的遗物→获得回血丹×1 + 灵石×30。走到一半时头顶掉下一只早已死透的吸血蚊群——尸体已经风干，但腹部鼓胀。剖开→获得灵草×1（它死前最后吸的是一株灵草的血）。
2. 走右路（宽干新路） — **结果**：战斗概率高，但有巡逻队。**下一层事件池偏移**：精英怪权重×2，但宝箱概率+10%。通道中途遇到正在巡逻的骸骨兵×2——立即进入战斗（替换下一层的roll结果）。击败后在灯台暗格中发现古币×5 + 灵石×50。那些灯台是新添的——有人在定期巡逻这条通道。你不想知道是谁。
3. 原地观察再决定 — **结果**：消耗时间观察两条路的痕迹。发现右路地面有新鲜脚印——不是人类的。左路深处传来微弱的风声，说明有出口。**下一层无偏移**，但本层获得信息buff→下场战斗中敌方先手-1回合（你的队伍提前洞察了环境）。观察期间在岩壁夹缝中发现解毒丹×1。

#### 事件：上下分层
**描述**：通道在此处被一道巨大的断层劈开——你正站在断层边缘。上层是天然岩洞穹顶，钟乳石密布，藤蔓从裂隙垂落，可以攀爬而上。下层是断裂的石阶，往下延伸没入黑暗，石阶上散落着明显不属于这个时代的物品：一面锈蚀的盾牌、半卷未烧尽的符箓、还有一个尚且完好的青铜香炉。

**选项**：
1. 爬上岩洞穹顶 — **结果**：攀爬路线，避开地面巡逻。进行一次SPD判定（全队平均SPD vs 难度90）。成功→**跳过本层所有战斗**，下一层直接进入。途中采到岩壁上的聚灵草×3 + 钟乳石缝隙中的解毒菇×2。失败→1名后排成员坠落（ATK×40%伤害），但仍成功到达上层。**下一层事件池偏移**：空层权重+15%（你在高处，能提前看到危险区域）。
2. 沿断裂石阶下行 — **结果**：深入古建筑下层。捡起沿途的遗物→获得青铜香炉（可出售灵石×60）+ 铁矿石×3。但下行越深，空气越沉闷。**下一层事件池偏移**：精英怪权重+20%（你正在接近某个核心区域）。捡到的那半卷符箓上写着一个「封」字——它没烧完，说明封印没完成。
3. 原地用藤蔓降至下层再上行 — **结果**：上下都探，但耗时翻倍。获取两边的资源（聚灵草×2 + 青铜香炉 + 灵石×30），但**本层结算后，下一层敌方全体获得「警觉」状态**（首回合DEF+15%）。你在藤蔓上留下了一条清晰的攀爬痕迹——后来的东西会知道你去了哪里。

#### 事件：水路旱路
**描述**：通道尽头被水淹没了一半。左侧是没膝的黑水，水面不时泛起不自然的波纹——水下有东西在动，但看不清是什么。右侧是露出水面的石堤，干燥可行，但石堤上每隔几步就有一堆干枯的骸骨，被排列得整整齐齐，头骨全部朝向一个方向。

**选项**：
1. 涉水而行（左路） — **结果**：水下潜伏着潮汐海妖×1——进入战斗。水战中所有非水属性技能伤害-10%。胜利后获得海妖内丹×2 + 珍珠×1。但全队在战斗中站位被潮汐打乱——**下场战斗开始时，你的前后排站位随机交换**（1↔3、2↔4等，仅影响第1回合）。**下一层事件池偏移**：宝箱权重+15%（水下冲刷出许多原本埋在淤泥里的东西）。

#### 事件：天道碎片
**描述**：半空中悬浮着一块发光的碎片——约莫手掌大小，形状不规则，边缘锋利如断口。它不是灵石，不是法器，不是任何你能叫出名字的东西。它只是在那里，散发着让人心悸的纯净光芒。空气在它周围扭曲，地面上的碎石在距它三尺处自动悬浮。它是一块天道法则的碎片——上古天都毁灭时，天道本身也被击碎了。这些碎片散落在遗迹各处，每一块都蕴含着改变现实的力量。

**选项**：
1. 尝试吸收碎片 — **结果**：全队中道心最高的成员进行道心判定（难度180）。成功→该成员吸收天道碎片，获得永久buff「天道共鸣」→全属性+3%（即使在副本外也生效，但每人仅限1次）。失败→天道碎片的力量超出了承受范围→该成员受到重创（HP-50%），但碎片的一部分仍被吸收→获得临时buff「天道余韵」→本次副本中ATK+20%。无论成败→吸收过程中逸散的能量凝结为灵石×300 + 鸿蒙紫气×1。
2. 封印碎片带走 — **结果**：需太乙道宗成员或持有封印类法器。成功封印→获得「封印的天道碎片」（特殊道具，可在村子中用于藏经阁Lv5解锁一门特殊功法或用于装备附魔）。封印中碎片不断挣扎→搬运者受到ATK×15%的反噬伤害。若无封印能力→强行搬运→碎片在手中爆炸，全队受到ATK×60%真实伤害（无视DEF），但爆炸后留下天道碎片×5 + 上古灵石×3。
3. 不碰碎片，绕道而行 — **结果**：你克制了贪婪。安全通过。碎片在你经过时轻轻鸣响——不是诱惑，是赞赏。全队道心+8。经过碎片后，发现前方路上有一盏被碎片光芒重新点亮的古灯→灯的底座刻着「天都·通明殿」，灯亮着意味着通明殿的余晖仍在。获得buff「通明灯引」→本次副本剩余层数中，每层的事件类别提前知晓（在roll事件前先看到该层会触发哪个类别）。古灯光芒中掉落→获得灵石×80 + 清心丹×2。

#### 事件：空间裂隙
**描述**：空气中撕开了一道裂缝——不是比喻。空间的裂口边缘像烧焦的纸一样卷曲，裂缝内部充满了旋转的星云和无法理解的色彩。裂缝在缓慢地扩张和收缩，像在呼吸。裂缝的四周散落着不属于这个时空的碎片：一枚刻有未来日期的铜币、一把材质无法辨认的断剑、还有一页写着明天日期的日历。裂缝的另一端是未知——可能是天都的另一个角落，也可能是三百年前的天都，也可能是三百年后的尸骸村。

**选项**：
1. 穿过空间裂隙 — **结果**：全队跳入裂隙→被传送到副本的**随机一层**（可能是已通过的层也可能是未通过的层，若传送到已通过的层则改为**跳过本层，直接进入下一层**，并获得探索奖励）。传送过程中受到空间压力→全队HP-20%。但裂隙中的空间能量凝结在身上→获得混沌晶核×2 + 法则结晶×1。传送的终点是一个意外的地点→额外获得该地点的特殊物品（如在东境获得珍珠×3，在中州获得天道碎片×3）。
2. 封闭裂隙 — **结果**：太乙道宗成员可以尝试封堵空间裂隙。消耗道印×2→裂隙被临时封印。封印过程中裂隙剧烈收缩，将内部最后喷出的能量物质抛射出来→获得灵石×200 + 陨铁×2。若无太乙道宗成员→尝试暴力封堵→裂隙爆炸，全队受到ATK×40%伤害但裂隙被炸塌（安全了）。爆炸把裂隙中的东西炸了出来→获得上古灵石×3 + 残破功法卷轴×5。
3. 从裂隙中抓取物品 — **结果**：不进入裂隙，只伸手进去捞东西。进行一次道心判定（取全队最高道心 vs 难度130）。成功→捞出3件物品（从五域掉落池中随机抽取，品质至少为蓝色）。失败→手臂被空间乱流割伤（损失HP-15%），但至少捞出了1件物品。无论成败，这种行为很不明智——代价不是伤在手上，而是被裂隙「记住了」。**本次副本BOSS战中，有一个裂隙会在战斗中短暂开启，把一名随机队员拖入虚空1回合**（无法行动，但返回时带着1件额外战利品）。

## 附录：事件机制速查

### A. 事件类别权重表

| 类别 | 权重 | 设计意图 |
|------|:---:|----------|
| 宝箱 | 20% | 最常见的"奖励"事件——有风险，有回报 |
| 陷阱 | 15% | 消耗队伍资源的考验——丹药、HP、站位 |
| 岔路 | 15% | 改变后续副本结构的节点——真正的策略选择 |
| 奇遇 | 10% | 稀有而有叙事深度的遭遇——道心考验为主 |
| 休息点 | 10% | 回复资源的喘息机会——在五域中弥足珍贵 |
| 精英怪 | 15% | 高难度高回报的战斗——检验队伍硬实力 |
| 空层 | 15% | 叙事驱动的"平静"时刻——铺垫氛围，积蓄情绪 |
