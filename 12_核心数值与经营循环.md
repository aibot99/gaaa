# 核心数值与经营循环

> **P0优先级** — 本文档定义了游戏从「设定」到「可实现」所需的三大基础系统。

## 一、修仙者属性体系

### 1.1 基础属性

每位修仙者有4项战斗属性 + 1项特殊属性：

| 属性 | 英文 | 说明 |
|------|------|------|
| 生命值 | HP | 归零则死亡，留下尸体 |
| 攻击力 | ATK | 影响所有伤害/治疗技能效果 |
| 防御力 | DEF | 减免受到的物理/魔法伤害 |
| 速度 | SPD | 决定行动条累积速度（每回合SPD累加，满100行动） |
| 道心 | Dao | 独立系统，见09_道心系统.md，初始100，不受等级影响 |

```

│ 阶位 │ 名称   │  HP  │ ATK  │ DEF  │ SPD  │ 升阶所需经验 │
├──────┼────────┼──────┼──────┼──────┼──────┼──────────────┤
│  1   │ 练气   │  250 │  35  │  18  │  12  │     —       │

**设计说明：**
- HP增长约1.5~1.55×/阶，天仙HP是练气的26倍
- ATK增长约1.45×/阶，天仙ATK是练气的18倍
- DEF增长约1.4×/阶，天仙DEF是练气的12.5倍
- SPD增长2~3点/阶，高阶先发优势显著但不至于碾压
- 相邻阶位单挑，高阶有约40~50%优势——符合修仙中「跨境如隔山」但不至于完全碾压

### 1.3 个体差异

同一阶位的修仙者，基础属性在±15%范围内随机浮动：

| 差异标签 | 浮动 | 概率 |
|----------|------|------|
| 天资卓绝 | +10%~+15% | 5% |
| 根骨上佳 | +5%~+10% | 15% |
| 资质平平 | -5%~+5% | 60% |
| 先天不足 | -10%~-5% | 15% |
| 经脉残缺 | -15%~-10% | 5% |

此外，每个修仙者可拥有最多2条随机**个人特质**（如「剑痴: 剑系伤害+10%」「药体: 受到治疗+20%」「胆怯: 道心下降翻倍」等），特质池后续文档补充。

## 二、战斗数值公式

### 2.1 核心伤害公式

```
原始伤害 = 攻击力 × 技能倍率
减伤率   = 防御力 / (防御力 + 200)
最终伤害 = 原始伤害 × (1 - 减伤率)  且不低于 原始伤害 × 15%
```

**验证示例：**

| 攻击方 | 防御方 | 原始伤害(×1.0) | 减伤率 | 最终伤害 |
|--------|--------|---------------|--------|----------|
| 练气ATK35 | 练气DEF18 | 35 | 8.3% | 32 |
| 金丹ATK75 | 金丹DEF36 | 75 | 15.3% | 64 |
| 化神ATK155 | 化神DEF68 | 155 | 25.4% | 116 |
| 天仙ATK620 | 天仙DEF225 | 620 | 52.9% | 292 |
| 天仙ATK620 | 练气DEF18 | 620 | 8.3% | 568（碾压，基本秒杀） |
| 练气ATK35 | 天仙DEF225 | 35 | 52.9% | 16.5（刮痧但非零） |

### 2.2 伤害类型

| 类型 | 计算方式 | 说明 |
|------|----------|------|
| 物理/法术伤害 | 标准公式 | 受DEF减免 |
| 真实伤害 | ATK × 技能倍率（无视DEF） | 直接扣血 |
| 百分比伤害 | 目标最大HP × % | 不参与伤害公式，直接扣除 |
| 治疗 | ATK × 治疗倍率 | 不受DEF影响 |

### 2.3 暴击系统

```
基础暴击率 = 5%
暴击伤害   = 150%（即1.5倍）
```

- 「必定暴击」技能跳过暴击率判定，直接触发
- 部分技能/装备/特质可提升暴击率和暴击伤害

### 2.4 命中与闪避

```
基础命中率 = 100%
闪避率     = 0%（除非技能/装备提供）
最终命中率 = 基础命中率 + 命中修正 - 目标闪避率  （上限100%，下限30%）
```

- 「无法被选中」(如灵鹰) = 单体技能无法指定，但全体技能仍然命中
- 「隐身」 = 单体技能不可指定，全体技能50%概率命中

### 2.5 元素属性

五域副本中涉及元素克制。基础元素体系：

```
┌─────┬─────────────────────────────────┐
│ 元素 │ 克制（伤害+30%） 被克（伤害-30%） │
├─────┼─────────────────────────────────┤
│  金  │ 木              │ 火             │
│  木  │ 土              │ 金             │
│  水  │ 火              │ 土             │
│  火  │ 金              │ 水             │
│  土  │ 水              │ 木             │
├─────┼─────────────────────────────────┤
│  雷  │ 水（额外+20%）  │ 土（额外-20%） │ (金木变体)    │
│  冰  │ 火（额外+20%）  │ 火（额外-20%） │ (水变体)      │
└─────┴─────────────────────────────────┘
```

门派技能默认**无属性**（不受克制影响），除非技能描述中明确标注属性（如「雷属性伤害」「火属性伤害」）。

### 2.6 DOT（持续伤害）结算规则

```
DOT伤害在目标行动结束时结算（每回合一次）
同名DOT叠加：刷新持续时间，层数叠加（上限按技能说明）
异名DOT共存：各自独立结算
DOT不会被DEF减免（本质是百分比或固定值伤害）
```

### 2.7 控制效果标准化

| 类型 | 效果 | 可驱散 | 示例技能 |
|------|------|--------|----------|
| 眩晕 | 跳过回合 | ✓ | 天罡镇魔(击退失败) |

| 缠绕 | 同定身 | ✓ | 毒丝缠绕 |

| 沉默 | 无法使用主动技能 | ✓ | 镇魂真言 |
| 混乱 | 随机攻击任意目标 | ✓ | 极乐世界 |

- BOSS在阶段三免疫部分控制（按BOSS说明）

```
         ┌──────────────────────────────────────┐
         │                                      │
         ▼                                      │

   │ 地牢冒险  │ ────────────→  │ 城镇建设  │    │
   │ (消耗1年) │               │ 建筑升级  │    │

         │                         │           │
         ▼                         ▼           │

   │ 修仙者成长 │               │ 招募/培养 │    │
   │ 升阶突破  │ ←──────────── │ 更强队友  │    │

         └─────────────────────────┘           │

                     └─────────────────────────┘

### 3.4 经验与升阶

#### 3.4.1 副本经验获取

每次冒险结束后，存活队员获得经验。经验总量 = 基础经验 × 难度系数 × 任务系数，**平均分配**给所有存活队员。

| 区域 | 基础经验 | 难度系数 |
|------|----------|----------|
| 东境 | 80 | 普通×1.0 / 困难×1.5 / 噩梦×2.5 |
| 南疆 | 180 | 同上 |
| 西域 | 350 | 同上 |
| 北境 | 600 | 同上 |
| 中州 | 1000 | 同上 |

任务系数：★ = 0.5 | ★★ = 1.0 | ★★★ = 1.5 | ★★★★ = 2.0 | ★★★★★ = 3.0

**示例**：南疆★★★任务·普通难度，4人队存活
- 总经验 = 180 × 1.0 × 1.5 = 270
- 每人 = 270/4 = 67.5 ≈ 68 XP

#### 3.4.2 升阶条件

```
经验达标 + 当前阶位满经验 → 可尝试升阶
升阶基础成功率 = 80%
洞府加成：+5%~15%（按等级）
破境丹加成：+20%
最终成功率 = min(95%, 基础成功率 + 各类加成)
```

升阶失败：保留80%经验（不掉阶），可再次尝试。

#### 3.4.3 天仙之上

天仙（9阶满级）后：
- 经验继续累积但不再升阶
- 天仙修仙者可进入「天道感悟」状态——每积累5000经验获得1点「天道感悟」
- 集齐5点天道感悟 + 天道丹 + 登天令 → 触发飞升结局
- 飞升后该修仙者离开队伍，在「飞升碑」上留名
- 此为游戏终极目标之一：**培养至少1名修仙者飞升**

## 四、公式速查表

### 4.1 伤害完整结算流程

```
1. 计算原始伤害 = ATK × 技能倍率
2. 检查元素克制：±30%
3. 检查暴击：5%基础 + 暴击率修正 → 暴击则 ×1.5
4. 计算减伤率 = DEF/(DEF+200)
5. 最终伤害 = 原始伤害 × (1 - 减伤率)，不低于15%
6. 应用各类增减益（护盾吸收、减伤BUFF等）
7. 扣除HP
```

### 4.2 治疗结算流程

```
1. 计算原始治疗 = ATK × 治疗倍率
2. 应用治疗加成（药力层数等）
3. 应用治疗衰减（重伤等DEBUFF）
4. 回复HP（不超过最大HP）
```

### 4.3 行动条结算

```
每回合开始时，所有存活单位 SPD 加入行动条
行动条 ≥ 100 的单位获得行动权
行动后行动条 -100（余数保留）
先满100者先行动，同满则SPD高者优先
```
