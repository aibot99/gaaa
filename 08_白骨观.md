# 白骨观（隐藏门派）

| 门派 | 白骨观 |
| 定位 | 隐藏门派（尸骸村深处的禁忌传承） |
| 站位偏好 | 3,4,5号位 |
| 资源 | 死气与白骨 — 双资源系统。死气：上限10层，友方或敌方单位死亡+2层，尸体被摧毁+1层，使用死灵技能消耗。白骨：无上限，消耗自身10%最大生命获得1根白骨，白骨用于骨系技能。死气增幅死灵伤害（每层+7%），白骨提升骨系技能硬度（每根+5%效果）。 |
| 核心机制 | 后排死灵法师，利用尸体和自残获取资源，死气白骨双修 |
| 流派1 | 死灵主宰 — 死灵操控流派，利用战场上每一具尸体作为武器，尸体越多力量越强，是真正的战场收割者。 |
| 流派2 | 白骨修士 — 白骨战斗流派，以自残换取白骨强化自身，化骨为甲为刃，是攻防一体的近战斗士。 |
| 尸体联动 | 白骨观是唯一与尸体系统深度绑定的门派——引爆尸体、复活尸体为己用、利用死气增强技能 |
| 特色 | 战斗越惨烈越强——敌我阵亡皆为养料，是站位机制下的最强后期门派 |
