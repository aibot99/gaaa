# 运营核心循环详解

> 本文档从玩家视角详细描述「模拟经营」侧的完整玩法循环——每一年你做什么、面对什么选择、各系统如何交织。

---

## 一、循环节奏：一年一回合

游戏的核心节奏是**回合制经营**：

```
┌─────────────────────────────────────────────────┐
│                  [一年] 的完整流程                  │
│                                                   │
│  ① 年初：城镇生产结算                              │
│     ├─ 灵田/药园自动产出（灵石+药材）                │
│     ├─ 醉仙楼/百宝阁收入到账                         │
│     ├─ 仓库/粮仓腐败结算                            │
│     ├─ 修仙者被动恢复（道心自动回满、伤势恢复）        │
│     └─ 建筑升级完成（如果去年开工了）                 │
│                                                   │
│  ② 年中：管理决策阶段                              │
│     ├─ 查看村口新到的修仙者（收留/驱逐）              │
│     ├─ 升级建筑（花费灵石+时间）                     │
│     ├─ 炼器阁打造装备                                │
│     ├─ 炼丹房炼制丹药                                │
│     ├─ 百宝阁购买消耗品                              │
│     ├─ 藏经阁学习功法                                │
│     └─ 藏经阁尝试残卷合成                            │
│                                                   │
│  ③ 年末：选择冒险                                  │
│     ├─ 选择副本区域（东/南/西/北/中）                │
│     ├─ 从4个随机任务中选1个                          │
│     ├─ 配置队伍（1~5人+宠物）                        │
│     ├─ 分配装备和丹药                                │
│     └─ 出发 → 进入地牢冒险（消耗1年）                │
│                                                   │
│  ④ 冒险归来：即时结算（触发新一轮年初）              │
│     ├─ 任务奖励（灵石+经验+掉落）                    │
│     ├─ 阵亡/存活判定                                │
│     ├─ 道心恢复至100                                 │
│     └─ 村口新一批修仙者到来                          │
│                                                   │
└─────────────────────────────────────────────────┘
```

**关键设计意图**：经营决策在「年中」集中完成，玩家可以反复调整、比较选项、精打细算。一旦选择冒险出发，这一年就结束了——回来又是一轮新的年初结算。

## 二、年初：被动收入与腐败

每年的第一件事是看账本。

### 2.1 被动收入一览

```
◆ 城镇收入报告 ◆

灵田 Lv2         +60 灵石
药园 Lv1         +40 灵石
醉仙楼 Lv1       +25 灵石
百宝阁 Lv1       +35 灵石
─────────────────────
合计            +160 灵石

仓库腐败：损失 3 份灵谷（腐败率10% × 粮仓-70% = 3%）
```

玩家看到这个报告后，知道兜里有多少钱可以做今年的事。

### 2.2 腐败的决策压力

食物资源（灵谷、药材、特殊食材）会随时间腐败。这制造了一个**隐性压力**：

- 囤积资源 waiting for 更好的出售时机 → 不行，腐败会让你亏
- 必须在腐败和「现在用掉」之间做权衡
- 粮仓升级可缓解，但需要大量灵石投入——短期钱荒 vs 长期省心

**典型困境**：你有30份灵谷，仓库腐蚀率10%。是现在出售（每份5灵石=150灵石），还是等炼器阁Lv3解锁后做成丹药再卖（预估翻倍）？等1年损失3份（15灵石），但做成丹药可能多赚150灵石。算下来值得等——但如果今年急需灵石升级茅屋招募新人呢？

## 三、年中：管理决策阶段

这是玩家花时间最多的环节——没有时间限制，可以反复斟酌。

### 3.1 村口：招募决策

每次冒险归来，村口会刷新一批修仙者。这是唯一的人员补充渠道。

```
◆ 村口告示 ◆

1. [练气] 玄明 — 凌霄剑宗 — 资质平平
   "逃到此地，只求一处安身之所。"
   → [收留] [驱逐]

2. [练气] 青萝 — 悬壶医谷 — 根骨上佳
   "我能治病，换一口饭吃。"
   → [收留] [驱逐]

3. [筑基] 血手 — 幽冥血宗 — 天资卓绝
   "别问老子的过去。有架打就叫我。"
   → [收留] [驱逐]
```

**决策维度**：

| 考虑因素 | 说明 |
|----------|------|
| 门派 | 与现有队伍的契合度？有没有敌对门派冲突？ |
| 阶位 | 筑基比练气强一档，但招募后需要配得上他的副本 |
| 个体差异 | 「天资卓绝」稀有，错过可惜——但幽冥血宗跟你的太乙道宗队友合得来吗？ |
| 人口上限 | 茅屋/洞府/醉仙楼总容纳量。满了就必须驱逐旧人才能招新人 |
| 直觉 | 喜欢这个角色的人设吗？ |

**驱逐的后果**：被驱逐的修仙者会记恨。未来可能在副本中以敌人身份出现，或者在其他事件中带来麻烦（道心动摇的队友提到「那个被赶走的……」）。

### 3.2 建筑升级：何时升什么？

这是经营中最核心的资源分配博弈。假设你有300灵石：

```
选项A：灵田 Lv1→Lv2（100灵石，年收入+30）→ 3.3年回本
选项B：百宝阁 Lv1→Lv2（200灵石，解锁清心丹，年收入+35）→ 5.7年回本
选项C：攒着等炼器阁Lv1（200灵石，开启装备系统）
选项D：什么都不升，拿去买丹药给下次冒险用
```

**没有绝对正解**，取决于你的处境：

| 处境 | 推荐 |
|------|------|
| 队伍战损严重，道心总崩 | 百宝阁Lv2解锁清心丹 |
| 队伍稳定，但灵石收入低 | 灵田/药园先升 |
| 前排坦度不够，经常死人 | 攒钱升炼器阁打造防具 |
| 茅屋满了招不了新人 | 升茅屋（最便宜的扩容） |

**升级还消耗时间**：Lv3以上建筑需要1~2年升级期。这意味着你这一年花了钱但**暂时没有收益**——升级中的建筑不产出。大量建筑同时开工可能让一年的被动收入骤降。

### 3.3 藏经阁：功法选择

藏经阁Lv1解锁第一本功法。玩家需要选择给谁学、学什么：

- 通用功法（ATK+5%）给主力输出
- 但藏经阁Lv2才解锁流派2技能，所以早期每个修仙者只能用流派1的4个技能
- 从Lv2开始，修仙者可以**双流派切换**（战斗中不可切换，冒险前选择）

这是一个逐步解锁修仙者深度的节奏设计。

### 3.4 炼丹房与百宝阁：消耗品准备

这是冒险前的「采购」环节：

| 丹药 | 来源 | 费用 | 效果 | 携带上限 |
|------|------|------|------|----------|
| 回血丹 | 百宝阁/炼丹房 | 30灵石 | 战斗中使用，回复20%HP | 每人1颗 |
| 清心丹 | 百宝阁Lv2 | 50灵石 | 战斗中道心+20 | 每人1颗 |
| 解毒丹 | 炼丹房Lv2 | 40灵石 | 清除中毒 | 每人1颗 |

**购买决策**：去南疆（毒多）→ 多带解毒丹。去北境（控制多）→ 多带清心丹防道心崩溃。去东境（新手区）→ 少带甚至不带，省灵石。

这是一个让玩家根据副本特点**针对性准备**的系统，增加策略深度。

## 四、年末：选择冒险

### 4.1 副本选择逻辑

```
已解锁副本：
[1] 东境·万妖海域  推荐练气~金丹    （已通关普通2次，困难已解锁）
[2] 南疆·瘴气林海  推荐筑基~元婴    （已通关普通1次）
[3] 西域·黄沙古漠  推荐金丹~化神    （🔒 队伍平均≥金丹解锁）
```

玩家需要在**收益**和**风险**间权衡：

- 东境困难：稳定收益，但经验少，队伍练气/筑基成长慢
- 南疆普通：经验多、掉落好，但毒系怪物可能让治疗压力翻倍
- 西域未解锁：激励玩家培养队伍到金丹

### 4.2 任务选择

每个副本随机刷4个任务。玩家需要考虑：

- **任务类型**：讨伐（战斗密集）vs 采集（战斗少但耗时）vs 除魔（BOSS战）
- **特殊条件**：护送任务「NPC死亡即失败」——你的队伍保护能力够吗？
- **奖励侧重**：缺灵石选高灵石任务，缺经验选高经验任务，缺装备材料选狩猎
- **隐藏风险**：「除魔·镇海蛟龙」★★★★★ — BOSS战不可撤退，灭队血本无归

**经典博弈**：队伍状态不好（有人残血、道心偏低），但刷出了一个★★★★★除魔任务（传说道具掉落）。赌一把可能翻身，也可能全员阵亡损失一年收益。

### 4.3 队伍配置

选择5人+宠物出战。站位分配是关键决策：

```
你的队伍：
  前排(1上/2下)：需要坦克/血战型 → 凌霄剑宗/幽冥血宗/万兽山庄
  后排(3上/4中/5下)：需要输出/治疗/控制 → 太乙道宗(4号位)/悬壶医谷(4号位)
  宠物位(1个)：万兽山庄的召唤物 或 其他宠物
```

**配置张力**：
- 4号位（中央后排）最安全 → 放治疗（悬壶医谷）或核心辅助（太乙道宗）
- 1号位（前上）最危险但技能往往最强 → 放你的最强前排
- 门派好恶暂未实装，但未来同队凌霄剑宗+幽冥血宗可能有负面效果

## 五、冒险中：经营与战斗的交织点

冒险中的资源管理将经营侧的准备转化为实战：

### 5.1 丹药使用的时机博弈

你在百宝阁买了3颗回血丹，分配给前排的凌霄剑宗和悬壶医谷。

- 悬壶医谷血量危急时——用还是不用回血丹？她自己的治疗技能可能更好，用丹药可能浪费。但如果她被秒了，后面更难打。
- 前排凌霄剑宗被BOSS暴击，残血——立刻用还是等悬壶医谷的治疗？治疗有CD，可能来不及。

### 5.2 尸体与道心的连锁

幽冥血宗的「血煞夺魂」伤害随尸体数量增加。但队友死亡会引发：
1. 全体道心-15
2. 道心崩溃可能传染

→ 你是否故意不救某个队友，让他变成尸体供幽冥血宗/白骨观利用？这是道德困境，也是战术抉择。

### 5.3 撤退与全灭

某些任务允许撤退（如采集、侦查），有些不允许（除魔、守护）。
- 允许撤退的任务：见势不妙可以跑，保住队伍和部分奖励
- 不允许撤退的任务：要么赢，要么全灭，损失所有

这一点在选择任务时就已注定——选任务时的信息不对称本身就是玩法。

## 六、归来：结算与新一轮循环

冒险归来 → 奖励到账 → 道心回满 → 村口新人刷新 → 又是新一年的年初结算。

### 6.1 长期节奏演变

```
阶段1：挣扎求生（前5年）
  - 队伍：2~3名练气
  - 副本：东境来回刷
  - 灵石：年收入100~200，刚好够升1个建筑
  - 重心：活下来、扩容、攒第一个筑基

阶段2：站稳脚跟（5~15年）
  - 队伍：3~4名筑基~金丹
  - 副本：南疆+东境困难
  - 灵石：年收入300~500，每2~3年一轮建筑升级
  - 重心：装备起步、丹药储备、阵容深化

阶段3：割据一方（15~30年）
  - 队伍：4~5名元婴~化神
  - 副本：北境+西域困难
  - 灵石：年收入800~1500，可多线升级
  - 重心：稀有掉落、功法收集、天仙培养

阶段4：问道飞升（30年+）
  - 队伍：5名化神~天仙
  - 副本：中州噩梦
  - 灵石：年收入3000+，城镇满级
  - 重心：登天令、鸿蒙紫气、飞升结局
```

### 6.2 持久吸引力的来源

| 驱动力 | 说明 |
|--------|------|
| 数值成长 | 修仙者从练气250HP到天仙6500HP的跨越感 |
| 阵容构筑 | 10门派×2流派 = 20种build，队伍组合爆炸 |
| 建筑解锁 | 每一级建筑都有新功能，不是纯数值 |
| BOSS故事 | 每个BOSS有独立剧情，推进主线 |
| 飞升目标 | 遥远的终极目标，给所有经营赋予意义 |

## 七、典型案例：一次完整的年度决策

> 以下模拟玩家视角的一次年度操作。

**背景**：这是第6年。你有185灵石。队伍：1名金丹（凌霄剑宗·破军）、2名筑基（悬壶医谷·济世、太乙道宗·八卦）。建筑：灵田Lv2、药园Lv1、醉仙楼Lv1、百宝阁Lv1、茅屋Lv2、仓库Lv1、村口Lv1。刚完成东境困难副本，3人存活，道心良好。

**年初结算**：
```
灵田Lv2: +60
药园Lv1: +40
醉仙楼Lv1: +25
百宝阁Lv1: +35
合计 +160，余额 185+160=345
```

**村口刷新**：
```
1. [筑基] 铁岩 — 万兽山庄 — 根骨上佳
2. [练气] 毒牙 — 万毒门 — 资质平平
3. [练气] 明月 — 青丘狐族 — 天资卓绝
```

你的思考过程：
- 铁岩是前排，可以分担凌霄剑宗的压力——好
- 但万兽山庄的宠物位只有1个，现在还没宠物
- 明月「天资卓绝」稀有，但青丘狐族是后排，跟太乙道宗、悬壶医谷抢3/4/5号位
- 毒牙资质差，不考虑
- 茅屋Lv2可容纳5人，目前3人→还可以招2人

**决策**：收留铁岩。驱逐毒牙和明月（可惜了天资卓绝，但位置冲突）。

**年中管理**：
1. 给铁岩分配茅屋。
2. 百宝阁→Lv2（200灵石），解锁清心丹。余额345-200=145。
3. 剩下145灵石买3颗回血丹（90灵石）备下次冒险。

**年末冒险**：
- 南疆已解锁，刷出4个任务
- 选择「采集瘴气灵芝」（★☆☆☆☆），因为新来的铁岩等级低，不打硬仗
- 配置：前排(1:凌霄剑宗, 2:铁岩) 后排(3:太乙道宗, 4:悬壶医谷, 5:空)
- 铁岩在2号位，凌霄剑宗在1号位抗伤害，治疗在4号位——稳妥阵型

**冒险归来**：采集任务完成，150灵石+经验，无人阵亡。

→ 第7年开始，余额145+150=295灵石。铁岩获得经验。建筑有百宝阁Lv2。循环继续。

## 八、设计原则总结

1. **每个决策都有代价**：升级A就不能升级B，招这个人就不能招那个人
2. **信息不对称**：副本危险程度在选择前不完全可知，任务描述有「隐藏条件」
3. **短期 vs 长期**：买丹药（短期安全） vs 升建筑（长期收益）
4. **情感绑定**：亲手培养的修仙者战死，比数值损失更痛
5. **循序解锁**：新系统（装备、流派2、功法、飞升）按阶段敞开，不是开局全给
